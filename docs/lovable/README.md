эталон. Соответствие версий пакета ↔ контракта — в [../CHANGELOG.md](../CHANGELOG.md).

## Ключевые правила (жёсткие)

- **Design tokens only** — ни одного hex в компонентах; палитра заказов (8 цветов) провалидирована
  (colorblind + контраст), используется **вместе со штриховкой** (вторичное кодирование для ЧБ-печати).
- **i18n** de(дефолт)/ru — только ключи; числа/размеры — locale-формат; внутри всё в мм.
- **Desktop-first ≥1280px**, только светлая тема (MVP).
- **Публикация движка** — см. [../onboarding.md §6](../onboarding.md) (публикует пользователь, без otp).
