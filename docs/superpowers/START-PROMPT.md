## Подводные камни, добытые кровью (не наступай)

- **Проверка деплоя — по production-ref + смене имени бандла, НЕ по совпадению с локальным хешем** (Coolify
  собирает в другом окружении → другой content-hash при том же коде).
- **`densityFirst` ОБЯЗАН быть ≥ `strict`** — `packLoad` считает обе раскладки и берёт плотнейшую (`r0m`).
  Не сворачивай обратно в один регион.
- **Демо насыщено (~93% пола)** — свободные места в демо неустранимы упаковщиком (свойство габаритов). Замеряй
  плотность на ненасыщенном грузе, не на демо.
- **`backfillPockets` строго-добавляющий, той же ориентацией** → эталоны целы. Вращение в кармане может их сдвинуть.
- **Цвет заказа = стабильный `colorIndex`, не позиция.** Карта `orderId→colorIndex` персистится
  (`ladungsplaner.orderColors`) и прокидывается в Ladeplan. Порядок легенды/зон — по появлению, цвет — стабилен.
- **Демо транзиентно** (`skipNextSaveRef` + `result.transient`) — не персистить ни setup, ни load.
- **Строка позиции — `flex-wrap` (БЕЗ `xl:flex-nowrap`)**, имя `max-w-[14rem]`; длинные лейблы/контролы переносятся.
- **Кнопка вращения стопки — ВНУТРИ фигуры** (иначе некликабельна у краёв). **`InfoHint` умеет `align='right'`**.
- **Chrome НЕ печатает SVG `<pattern>`** — штриховка прямыми `<line>`/`<circle>` (`HatchMarks`). Не возвращать `<pattern>` в печать.
- **click-outside слушать `click`, НЕ `mousedown`** — иначе кнопка («+ Position») не срабатывает.
- **Конвенция вида сбоку:** больший `y` = ближний ряд (`depth 0`), меньший — дальше (тусклее). Поле `CutRect.rowY`.
- **`findGeometryViolations` пропускает пары одной колонки** — для ручных правок отдельная проверка футпринтов (`overlapsOtherStack`).
- **Ручной поворот — только yaw** (`lwh`↔`wlh`); якорь — угол `(x,y)` (целые мм).
- **`localStorage`-ключи:** `ladungsplaner.setup` (форма), `ladungsplaner.load` (план), `ladungsplaner.orderColors`
  (цвета заказов), `ladungsplaner.locale` (язык) — кэш черновика, НЕ источник истины.
  `ladungsplaner.palletPresets` — каталог пресетов, **переживает Reset**. `test-setup.ts` чистит localStorage в `afterEach`.
- **dist в .gitignore** — на чистой машине `npm ci` + сборка пакетов, иначе web/server-тесты не видят `@shadrin-v/engine`/`i18n`.
- **jsdom без layout** — баги вёрстки/геометрии указателя тестами не воспроизводятся; тест сторожит намерение.
- **`forkAccess:'twoSides'` под `combined`** — no-op (обе двери), физически верно (есть InfoHint-подсказка).
- **git checkout+merge+push — по отдельности, НЕ через `&&`** (хук авто-экспортит `.beads/issues.jsonl` при commit).
