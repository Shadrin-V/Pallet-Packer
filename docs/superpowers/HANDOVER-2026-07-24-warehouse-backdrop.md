## Что сделано (итог, а не путь)

`WarehouseBackdrop` — вид сверху на двор под разрезами кузова:
- **Пол — плоская заливка `#cbcdcd`** (const `FLOOR`, экспортируется). Ни узора, ни тайлинга.
- **Два дока-иллюстрации владельца** по краям: `warehouse-floor-left.png` (ящики/паллеты/зарядка),
  `warehouse-floor-right.png` (погрузчик носом ВВЕРХ к грузовику + стеллажи + разметка). PNG-**вырезки на
  прозрачном фоне** (`Lager_l`/`Lager_r` из Downloads), ложатся поверх заливки со своими тенями.
- Каждый док на **100% высоты пола**, масштаб по своей стороне (`w/h * height`, без искажений), прижат к краю.
- Всё в мм-координатах SVG склада → масштаб заперт к плиткам by construction.
- `ForkliftMark` удалён (заменён погрузчиком из ассета); логика «парковать сценерию в свободном углу» убрана.
- Груз на складе — `StackShape backing` (непрозрачная paper-подложка): фон не просвечивает сквозь 16%-тинт.
- Карточка склада — 100% под пол, без плоских strip'ов: SVG full-bleed, заголовок «LAGER» **над** карточкой
  (как ярлыки грузовика). Мин-высота склада = ширине кузова. Dropzone-пунктир убран.

## Решения / поздние развороты (важно — их было много)

1. **Финал ≠ первые версии.** Прошли: 3-slice+`<pattern>` тайл-узор → фикс-размер сценерии → 100%-высоты →
   **плоская заливка + доки-вырезки**. `close-reason` у `41e.5` описывает ПЕРВУЮ версию (3-slice/фикс) —
   к задаче добавлен `bd comment` с финалом. Исходный бриф `specs/2026-07-23-warehouse-scenery-asset-brief.md`
   помечен `⚠️ SUPERSEDED` (реализовано растром+плоским полом, не чистым вектором).
2. **Переменные пропорции → только плоская заливка.** Полоса-склад — прямоугольник переменной ширины
   (длина кузова) и высоты (глубина буфера). Фикс-пропорциональная картинка искажается/даёт поля; тайл
   растрового слайса оставлял sub-pixel швы (владелец их видел). Плоский тон бесшовен всегда; арт — по краям.
3. **Растр, не вектор** (память [[hand-svg-illustration-trap]]): присланные SVG — авто-трейс 3.29 МБ со
   встроенным растром, пользы над PNG нет. Плата: доки не перекрашиваются токеном темы (тёмная тема — НЕ
   делаем, пользователь сказал «оставь как есть»).

## Грабли

1. **`apps/server`-тесты локально КРАСНЫЕ** — `better-sqlite3 NODE_MODULE_VERSION` (ABI). Пре-существующее,
   в CI (чистый Node) зелёное. Корневой `npm test` = 638 (38 падают = server ABI). «Зелёный» этой сессии —
   `apps/web`+`packages` = **587**. Не чинить в рамках фич.
2. **«Убрать зазоры» ≠ «убрать узор».** Владелец просил починить sub-pixel швы тайла, я один раз ошибочно
   выкинул узор и нарисовал свою разметку — откатил. Урок: уточнять, «убрать X» — это X или дефект X.
3. **HTML-заголовок поверх SVG «моет» плитки.** Оверлей-скрим (paper-градиент) размывал верх первого ряда
   (px-mm рассинхрон). Решение: заголовок вынесен НАД карточкой, не поверх.
4. **i18n грузится из собранного `dist`** — правка словаря/ключа не видна тестам, пока не `npm run build -w
   @shadrin-v/i18n` (в этой сессии i18n не трогали, но правило живёт).
